Processing YAML files

Our Vite plugin can now recognize .yaml files that are being loaded. Next we’ll need to add logic for converting the .yaml files into Javascript and pass the output from load hook.

To get the content of requested file we’ll need to use readFileSync method from node:fs. We can convert the YAML content into Javascript using yaml package. It has a parse() function that takes YAML content as string.
